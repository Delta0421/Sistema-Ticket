# Sistema-Ticket
Descripción del proyecto

Este proyecto implementa un sistema de ticketing para la gestión de incidencias en el laboratorio de cómputo LAB-LIS. El sistema permite registrar incidencias, clasificarlas, asignar técnicos, consultar el estado de los casos y mantener un historial de acciones realizadas.

El diseño incluye clases como Incidencia, Usuario, Tecnico, RegistroHistorico y un manejador principal que administra las operaciones del sistema. El enfoque está orientado al uso de objetos, encapsulamiento, abstracción y relaciones entre clases, usando el lenguaje de programación Java.
